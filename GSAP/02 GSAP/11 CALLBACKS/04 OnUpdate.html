Progress: 0%

              Track animation progress

              🧠 What is happening?
                    onUpdate runs many time
                    this.progress() gives animation progress

                    Used for:
                    Progress bars
                    Counters
                    Scroll indicators
          
        📌 onUpdate
            Runs while the animation is running.
            Example:
                onUpdate: function () {
                  const percent = Math.round(this.progress() * 100);
                  console.log("Progress: " + percent + "%");
                }
