9️⃣ CALLBACKS
      📌 Concept
          Run code when animation starts/ends.
          “Run a function when something happens during animation”
          Just like:
                    After animation starts → do something
                    While animation is running → do something
                    After animation ends → do something
  
      📌 onStart
          Runs when the animation starts.
          Example:
              onStart: () => console.log("Started")

      📌 onComplete
          Runs when the animation completes.
          Example:
              onComplete: () => console.log("Completed")
  
    2️⃣ Common GSAP Callbacks List
        Callback	              When it runs
        onStart	                When animation starts
        onUpdate	              While animation is running
        onComplete	            When animation ends
        onRepeat	              Every time animation repeats
        onReverseComplete	      When reverse finishes
