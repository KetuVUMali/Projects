GSAP Timeline Navigation Example : Open and Close Menu

This example demonstrates a GSAP timeline used to animate a side menu opening and closing.

  • The menu slides in from the left when the "Open Menu" button is clicked.
  • The menu slides back out when the "Close Menu" button is clicked.
  • The timeline is paused initially and controlled via button clicks.

This showcases how timelines can be used for interactive UI animations.

            const menuTL = gsap.timeline({ paused: true });
        
            menuTL.to(".menu", {
                x: 200,                  // Slide menu in by 200px
                duration: 0.5,           // Animation duration of 0.5 seconds
                ease: "power2.out"       // Easing function for smoothness
            });
