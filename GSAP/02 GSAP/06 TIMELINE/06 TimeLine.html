GSAP Timeline with Controls Example : Timeline controls

This example demonstrates a GSAP timeline with play, pause, and reverse controls for a box animation.

  • The box first moves 300px to the right over 1 second.
  • Then, it rotates 180 degrees over 1 second.
  • Buttons allow you to control the timeline playback.

This showcases how timelines can be controlled interactively.

            tl.to(".box", { x: 300, duration: 1 })         // Box moves right for 1 second
                .to(".box", { rotate: 180, duration: 1 }); // Box rotates 180 degrees for 1 second
        
            🧠 Explanation (Step-by-step)
                        Timeline created (paused)
                        Box moves right for 1 second
                        After box1 finishes → box rotates 180 degrees
                        No animation overlaps
✅ Timeline Controls: tl.play() → Start or resume the timeline tl.pause() → Pause the timeline tl.reverse() → Reverse the timeline direction
