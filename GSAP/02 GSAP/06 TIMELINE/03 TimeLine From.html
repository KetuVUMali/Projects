Page load animation using timeline

GSAP Timeline Example

This example demonstrates a GSAP timeline that animates three boxes when the page loads.

  • Each box starts off-screen to the left (x: -200) and is fully transparent (opacity: 0).
  • They animate into view by moving to their original position (x: 0) and becoming fully opaque (opacity: 1).
  • The animations are staggered by 0.3 seconds, creating a sequential effect.

This showcases how timelines can be used for page load animations effectively.

            tl.from(".box", {
                x: -200,        // Start off-screen to the left
                opacity: 0,     // Fully transparent
                duration: 1,   // 1 second duration
                stagger: 0.3   // Stagger start times by 0.3 seconds
            });

🧠 Explanation (Step-by-step)

Timeline created

Box1 animates from left to center with fade-in

After 0.3s delay, Box2 animates from left to center with fade-in

After another 0.3s delay, Box3 animates from left to center with fade-in

No overlaps, smooth sequential animation on page load
