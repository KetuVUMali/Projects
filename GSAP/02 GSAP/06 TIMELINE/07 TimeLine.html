GSAP Timeline with Repeat and Yoyo Example : Looping timelines

This example demonstrates a GSAP timeline that continuously repeats and reverses the animation of a box.

  • The box first moves 200px to the right over 1 second.
  • Then, it scales up to 1.5 times its original size over 1 second.
  • The timeline is set to repeat indefinitely and reverse direction after each cycle.

This showcases how to create looping animations using GSAP timelines with repeat and yoyo properties.

    const tl = gsap.timeline({
        repeat: -1,                 // Repeat indefinitely
        yoyo: true                  // Reverse direction after each cycle
    });
    
    tl.to(".box", { x: 200, duration: 1 })         // Box moves right for 1 second
        .to(".box", { scale: 1.5, duration: 1 }); // Box scales up for 1 second
    
        🧠 Explanation
                    Box moves right for 1 second
                    Box scales up for 1 second
                    Timeline repeats indefinitely
                    Timeline reverses direction after each cycle
