GSAP Timeline with Absolute Positioning Example : Using absolute times

This example demonstrates a GSAP timeline where a box moves, then moves down, and finally scales up using absolute positioning for the animations.

  • The box first moves 200px to the right over 1 second, starting at time 0.
  • Then, it moves 100px down over 1 second, starting at time 1.
  • Finally, it scales up to 1.5 times its original size over 1 second, starting at time 2.

This showcases how timelines can be used to create animations with precise timing using absolute positions.

            tl.to(".box", { x: 200, duration: 1 }, 0)
                                                        // Box moves right for 1 second, starting at time 0
            .to(".box", { y: 100, duration: 1 }, 1)
                                                        // Box moves down for 1 second, starting at time 1
            .to(".box", { scale: 1.5, duration: 1 }, 2);
                                                        // Box scales up for 1 second, starting at time 2
        
            🧠 Explanation
                        0 → start at 0s
                        1 → start at 1s
                        2 → start at 2s
                        You manually control time
