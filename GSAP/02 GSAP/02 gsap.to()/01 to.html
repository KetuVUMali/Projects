gsap.to() Method : original to Animate Movement
The gsap.to() method animates an element from its current state to the specified properties.

Syntax: 
            gsap.to(target, {vars});

Example: 
            gsap.to(".box", {x: 200, duration: 1});
This moves the element with class "box" 200 pixels to the right over 1 second. 
        Explanation:
            gsap.to() → GSAP method to animate to specified properties
            .box is selected
            x: 200 → moves box 200px right
            duration: 1 → animation takes 1 second
            Animation runs as soon as page loads
