SVG path Curve :
                            1. Quadratic  Curve
                            2. Bezier Curve

        1. Quadratic Curve :
            Q x1 y1, x y
            Q -> command for Quadratic Curve
            x1 y1 -> control point
            x y -> end point

        2. Bezier Curve :
            C x1 y1, x2 y2, x y
            C -> command for  Bezier Curve
            x1 y1 -> first control point
            x2 y2 -> second control point

        Note :
                a cubic one, called with C, and a quadratic one, called with Q.
                stroke='' -> line color
                fill='' -> inside color

        1. Quadratic Curve Example :
            M 10 80 Q 95 10 180 80
            M -> move to   (Starting point) (10 : x-axis, 80 : y-axis)
            Q -> Quadratic Curve            (95 : x-axis, 10 : y-axis, 180 : x-axis, 80 : y-axis)
            
            (10,80) -> Curve Starting Line Point
            (95,10) -> Control Point (curve direction)(curve up-down)
            (180,80) -> Curve End Line Point
