Stagger : Stagger = delay animation of multiple elements one by one.
                  Group of elements appear one after another.
                  Group of elements apply same animation.
                  Multiple elements group appear one by one with some delay time called.


            syntax :
                    gsap.to(Selector, duration, animation)
                    
                    gsap.to(Selector, duration, animation).stagger(0.2)

            Example :
                    gsap.to(".card", {
                        y: 40,
                        opacity: 0,
                        stagger: 0.2,       // Stagger Animation Time
                        duration: 0.6
                    });

            Note : Stagger Animation Time = Delay Time

                    2. Stagger take positive value
                        Positive value = delay time
                        Display clockwise (Strath / Froward)

                    3. Stagger take negative value
                        Negative value = delay time
                        Display anticlockwise (Reverse / Backward)
