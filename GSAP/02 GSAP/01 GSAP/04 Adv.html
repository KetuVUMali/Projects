SCROLLTRIGGER INTERNAL WORKING :

    ScrollTrigger kya karta hai?

    STEP 1:
    Page load pe element ka position measure karta hai.

    STEP 2:
    Start & end points calculate karta hai.

    STEP 3:
    Scroll event listener attach karta hai.

    STEP 4:
    Scroll progress = (currentScroll - start) / (end - start)

    STEP 5:
    Is progress ko animation progress se map karta hai.

    Yahi reason hai scrub smooth hota hai.

6️⃣ SCRUB
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Meaning:
    Scroll ke sath animation sync.
    Scroll Trigger Scrolling sobat animation perform kart, scroll chya basis var 

Types:
    scrub: true → exact sync
    scrub: 2 → smooth 2 sec delay



SCRUB MATHEMATICAL LOGIC : 

    Without scrub:
        Animation independent hoti hai.

    With scrub:
        Scroll progress = animation progress.

    Example:
        Start: 100px
        End: 500px

    Scroll range = 400px

    Agar scroll 200px hai:
    Progress = 100 / 400 = 0.25

    Animation bhi 25% complete hogi.

    scrub: 2
        Means smoothing duration 2 seconds.

Use:
✔ Storytelling website
✔ Parallax effect
✔ Landing page sections



PIN DEEP UNDERSTANDING :

    Pin = element ko temporarily fixed position me daalna.

    Internally kya hota hai?

    STEP 1:
        Element ka placeholder create hota hai.

    STEP 2:
        Original element ko position: fixed diya jata hai.

    STEP 3:
        Scroll ke dauran element stuck rehta hai.

    STEP 4:
        End pe wapas normal flow me aa jata hai.

    IMPORTANT:
        Pin height calculation correct hona chahiye.
        Isliye images loaded hona zaruri.

SECTION 3: TRANSFORM VS TOP/LEFT
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Bad:

Good:
    x: 200

Why?

left → layout recalculation
x → transform matrix update

Transform = no reflow
Better performance
