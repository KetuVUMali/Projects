🚀 What is GSAP?
GSAP = GreenSock Animation Platform
👉 GSAP is a JavaScript animation library
👉 Used to create smooth, powerful, professional animations
🎯 Why use GSAP?
👉 High performance animations 👉 Easy to use API 👉 Works with all modern browsers 👉 Great documentation and community support 🔧 Basic GSAP Syntax
GSAP uses the `gsap.to()` and `gsap.from()` methods to create animations.
Example: Animate an element to move 100px to the right over 1 second. 
                gsap.to(".box", {duration: 1, x: 100});
