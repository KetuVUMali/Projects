Scroll More ⬇️
    👉 Explanation: Animate a box when it enters the screen

        🧠 WHAT IS HAPPENING?

            Page is scrollable
            Box is hidden initially
            When you scroll down and box enters viewport → animation starts

    
        ScrollTrigger Properties Used:
            trigger → element that triggers the animation
            start → when the animation should start (box top hits 80% of viewport)
            end → when the animation should end (box top hits 30% of viewport)
    
        👉 Animation starts when box reaches 80% viewport height
        👉 Animation ends when box reaches 30% viewport height

    
        Stop : Animation starts when the box's top edge reaches % of the viewport height.
        End : The box animates into view as you scroll down past the start point and completes the animation by the end point.
