        👉 Explanation: Staggered animation of multiple cards on scroll
                        Cards appear one-by-one when scrolling down the section


            🧠 WHAT IS HAPPENING?

                Page is scrollable
                Cards are hidden initially
                When you scroll down and section enters viewport → cards animate in one after another

    
        ScrollTrigger Properties Used:
            trigger → element that triggers the animation
            start → when the animation should start (section top hits 70% of viewport)

        GSAP Property Used:
            stagger → delay between the start times of each card's animation
