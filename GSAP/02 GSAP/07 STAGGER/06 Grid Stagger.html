GSAP Grid Stagger Example

This example demonstrates a GSAP stagger animation applied to a grid layout where multiple cells animate in a staggered manner.

  • Each cell starts 0.6 seconds after the previous one, following a grid pattern.
  • Cells scale up from 0 to their original size and fade in over 2 seconds.

This showcases how to create staggered animations in a grid layout using GSAP.

                🧠 Explanation (Step-by-step)
                    Select all .cell elements
                    Animate from opacity:0 and scale:0
                    Stagger each animation by 0.6s
                    Each animation lasts 2s
                    Create a grid layout with 2 rows and 3 columns
                    Start animations from the first cell
