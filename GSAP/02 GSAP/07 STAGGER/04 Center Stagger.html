GSAP Stagger Direction Example

This example demonstrates a GSAP stagger animation where multiple boxes animate from the center outwards with a delay.

  • Each box starts 0.2 seconds after the previous one, beginning from the center box to the first.
  • Boxes scale up from 0 to their original size over 0.5 seconds.

This showcases how to create staggered animations with direction control using GSAP.
