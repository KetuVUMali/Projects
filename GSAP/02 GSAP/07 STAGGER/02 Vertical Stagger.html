GSAP Stagger Basic Example

Animate multiple elements one by one with same animation.

This example demonstrates a simple GSAP stagger animation where multiple items animate one after another with a delay.

  • Each item starts 0.2 seconds after the previous one.
  • Items move up 50px and fade in over 0.5 seconds.

This showcases how to create staggered animations using GSAP for a sequential effect.

    🧠 Explanation (Step-by-step)
                    Select all .item elements
                    Animate from y:50 and opacity:0
                    Stagger each animation by 0.2s
                    Each animation lasts 0.5s
                    Creates a sequential entrance effect
    
        gsap.from(".item", {
                        y: 50,
                        opacity: 0,
                        duration: 0.5,       // Each animation lasts 0.5 seconds
                        stagger: 0.2        // Stagger start times by 0.2 seconds
                        });
