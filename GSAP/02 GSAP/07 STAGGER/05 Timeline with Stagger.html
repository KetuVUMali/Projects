GSAP Timeline with Stagger Example

This example demonstrates a GSAP timeline that animates multiple menu items with a staggered effect.

  • Each menu item slides in from the left and fades in.
  • There is a 0.15 second delay between the start of each item's animation.
  • The entire animation sequence is managed within a single timeline.

This showcases how to combine timelines and staggered animations using GSAP for a polished entrance effect.

        tl.from(".menu-item", {
            x: -100,
            opacity: 0,
            duration: 0.4,              // Each animation takes 0.4 seconds
            stagger: 0.15               // Stagger each animation by 0.15s
        });
        
            🧠 Explanation :
                            Timeline controls sequence
                            Stagger controls delay between items
                            Used in side menus, navbars
