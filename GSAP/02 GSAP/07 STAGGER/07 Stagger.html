GSAP Timeline Navigation with Stagger Example : Staggered menu animation

This example demonstrates a GSAP timeline that animates menu items with a staggered effect when opening and closing a menu.

  • Each menu item slides in from the left and fades in when the menu is opened.
  • There is a 0.15 second delay between the start of each item's animation.
  • The timeline is paused initially and controlled via button clicks.

This showcases how to combine timelines and staggered animations using GSAP for a polished menu animation effect.

        const navTL = gsap.timeline({ paused: true });      // Create paused timeline

        navTL.from(".nav div", {                            // Animate menu items with stagger
            x: -50,                                         // Start 50px left of the screen
            opacity: 0,                                     // Fade out initially
            duration: 0.4,                                  // Each animation takes 0.4 seconds
            stagger: 0.15                                   // Stagger each animation by 0.15s  
        });
