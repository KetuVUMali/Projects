GSAP Stagger Direction Example

This example demonstrates a GSAP stagger animation where multiple cards animate from the end towards the start with a delay.

  • Each card starts 0.2 seconds after the previous one, beginning from the last card to the first.
  • Cards scale up from 0 to their original size over 0.5 seconds.

This showcases how to create staggered animations with direction control using GSAP.

                🧠 Explanation (Step-by-step)
                    Select all .card elements
                    Animate from scale:0
                    Stagger each animation by 0.2s
                    Each animation lasts 0.5s
                    Start animations from the end towards the start

Stagger Direction : start, center, end, random

  • start : Start animations from the start towards the end.
  • center : Start animations from the center towards the end.
  • end : Start animations from the end towards the start.
  • random : Start animations in a random order
