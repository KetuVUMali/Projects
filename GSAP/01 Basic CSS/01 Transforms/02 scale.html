        2️⃣ scale transforms :

                scale means zoom.  (multiply size)

                scale(1) → normal size
                scale(1.5) → bigger
                scale(0.8) → smaller

                scaleX(1.5) → stretch width
                scaleY(0.5) → compress height

                scale() can take one or two values :
                    one value → scale both width and height
                    two values → first value scales width, second value scales height

                    ex : scale(1.5, 0.5) → width 1.5 times, height 0.5 times

                scaleX() : scales only the width of the element.
                scaleY() : scales only the height of the element.

     Note : Transforms do NOT affect surrounding elements.
