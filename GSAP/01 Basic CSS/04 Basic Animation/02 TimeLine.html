Timeline Animation : transition-delay
By using transition-delay, we can create a timeline animation effect where each element starts its animation at different times.
👉 Sequence of animations
👉 One animation after another


In this example:
  • The first box moves immediately when hovering.
  • The second box starts moving after a 0.3s delay.
  • The third box starts moving after a 0.6s delay.


CSS does NOT have real timeline like GSAP But we simulate using delays.
