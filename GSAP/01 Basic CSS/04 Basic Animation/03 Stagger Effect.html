Stagger Effect : transition-delay
By using transition-delay on multiple elements, we can create a stagger effect where each element animates in sequence.
👉 Elements appear one after another


In this example:
  • The first item moves in immediately when hovering.
  • The second item starts moving after a 0.2s delay.
  • The third item starts moving after a 0.4s delay.


👉 Same animation
👉 Starts one by one




duration : animation time delay : wait before start
