AOS = Animate On Scroll

        👉 It adds animation when elements enter the screen while scrolling.

        Real-life company usages :
                                    Website sections fade in smoothly
                                    Cards slide from left/right
                                    Content loads progressively
                                    Improves UX (User Experience)

        ⚠️ Important:
                        AOS is not heavy animation, it’s scroll-triggered animation.


            🔹 2. How AOS Works (Simple Logic)
                                1. When User scrolls page
                                2. AOS checks:
                                        Is element visible in viewport?
                                3.If YES → apply animation CSS classes to element
                                4. Animation runs once or multiple times

                        📌 Viewport = visible screen area


            🔹 3. Key Features of AOS
                                - Easy to use with data attributes
                                - Multiple animation types (fade, slide, zoom)
                                - Customizable duration, delay, easing
                                - Responsive and mobile-friendly
                                - Performance optimized for smooth scrolling
