Basic Concept: data-aos-offset

data-aos-offset controls WHEN animation starts during scrolling.

Offset means:
👉 Distance (in pixels) from bottom of screen
👉 Animation starts BEFORE element fully appears

Default Offset (120px)
Offset 50px (Late animation)
Offset 300px (Early animation)

Hero Section – Early Animation

Hero sections should animate EARLY so user feels smooth flow.
Large offset is recommended.

Hero Heading (offset: 400px)
Hero Subheading (offset: 350px)

About Section – Normal Scroll Timing

Informational content should animate when user is near it.
Default or medium offset works best.

About Text (offset: 120px)

Services Cards – Controlled Entry

Cards should animate only when user reaches them.
Medium offset avoids distraction.

Service 1 (150px)
Service 2 (150px)
Service 3 (150px)

Image / Media Section

Images should not animate too early.
User must SEE them first.

Image Animation (offset: 80px)

Statistics Section – Accurate Timing

Numbers should animate when user is ready to read them.
