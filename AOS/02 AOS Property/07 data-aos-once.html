What is data-aos-once?

data-aos-once decides:

👉 Should animation run ONLY ONE TIME 👉 OR run EVERY TIME user scrolls up & down

Think like notifications:
Show once ✔ | Repeat every time ❌

Scroll DOWN → then scroll UP → then scroll DOWN again

You will FEEL the difference clearly

Scenario 1: Hero Banner (Once = TRUE)

Hero sections should animate only ONCE.
Repeating animation looks unprofessional.

Welcome to Our Platform

This animation will NOT repeat on scroll

data-aos-once="true"

Scenario 2: Feature Cards (Once = FALSE)

Feature cards can animate every time.
Good for long pages and engagement.
