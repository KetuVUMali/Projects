data-aos-mirror="true"
→ Animate on scroll DOWN + UP

data-aos-mirror="false"
→ Animate only on scroll DOWN

What is data-aos-mirror?

data-aos-mirror controls animation behavior when you SCROLL BACK UP.

false → animation happens ONLY when scrolling DOWN true → animation happens BOTH DOWN & UP

Scroll down first, then scroll UP to see mirror behavior

Default Behavior (mirror = false)

Animation plays when element enters viewport.
When scrolling UP → animation does NOT replay.

✔ Most common & safe usage.

Only animates on scroll DOWN

Mirror Enabled (mirror = true)

Animation runs when scrolling DOWN
AND again when scrolling UP.

⚠ Use carefully – can feel repetitive.

Animates on DOWN & UP scroll

Reading Content (mirror = false)

Text content should NOT animate again when user scrolls back.

✔ Professional UX behavior.

Reading content – no reverse animation

Timeline / Story Section (mirror = true)

Timeline sections may animate both ways to reinforce storytelling.
