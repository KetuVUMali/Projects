What is data-aos-easing?

data-aos-easing controls HOW animation moves.

👉 It does NOT control time (duration) 👉 It controls motion feeling (speed curve)

Think like a car:
Slow start → Fast → Slow end = Smooth ride 🚗

Scroll down slowly and feel the difference

linear (No Smoothness)

Same speed from start to end.
❌ Feels robotic.
Rarely used in real projects.
