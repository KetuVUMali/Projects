Auto Col set 5 Column in every screen size

We wont to display mobile : 1 column, tablet : 2 column, desktop : 5 column.

row-cols : to create horizontal group of columns.
Then we use break point to manage column number in different screen size.
1. row-cols-1 : for mobile ( Mobile )
2. row-cols-sm-2 : for tablet ( Tablet )
3. row-cols-lg-5 : for desktop ( Desktop )
Note : break point use mostly all things like Grid System, Container, Row, Col, etc.


row-cols : It automatically set equal width to all columns in a row.
ex : 5 column = each column take 20% width of container


Explanation of break point used :
1. row-cols-1 : for extra small devices (<576px) : 1 column : Create 1 column in extra small devices
2. row-cols-sm-2 : for small devices (≥576px) : 2 column : Create 2 column in small devices
3. row-cols-lg-5 : for large devices (≥992px) : 5 column : Create 5 column in large devices
