Grid system

Use our powerful mobile-first flexbox grid to build layouts of all shapes and sizes thanks to a twelve column system, six default responsive tiers, Sass variables and mixins, and dozens of predefined classes.

Bootstrap's grid system is built with flexbox and allows up to 12 columns across the page.

The grid system is responsive, and the columns will re-arrange automatically depending on the screen size.
Make sure that the sum adds up to 12 or fewer (it is not required that you use all 12 available columns).

col = 12 / number of columns you want
col : every col take equal space with auto 12px padding (only left-right) to each column.
row : to create horizontal group of columns.
row : to create vertical group into columns.
row take full width of container with -12 margin automatically to adjust left-right padding of column.
col : +12, row : -12 balance each other. means col + row = 12 and take full width.
Auto colum : automatically set equal width to all columns in a row. (ex : 5 column = each column take 20% width of container)
