Bootstrap Image Alignment - All Examples

📌 What is Image Alignment?
    Image alignment means placing an image
    - Left
    - Center
    - Right
    - Responsive inside grid
    - Vertical & horizontal center

📌 Important Rule
    Always use 'img-fluid' for responsive images.

1️⃣ Normal Image (Default)

📌 Concept
    Image appears in normal flow (left by default)
    No alignment class used

2️⃣ Left Aligned Image (float-start)

📌 Concept
    float-start aligns image to the left

3️⃣ Right Aligned Image (float-end)

📌 Concept
    float-end aligns image to the right

4️⃣ Center Aligned Image (mx-auto)

📌 Concept
    d-block makes image block element
    mx-auto centers image horizontally

5️⃣ Center Image using text-center

📌 Concept
    Image is inline by default
    text-center centers inline elements

6️⃣ Image Alignment inside Grid

📌 Concept
    Use row and col for responsive layout
