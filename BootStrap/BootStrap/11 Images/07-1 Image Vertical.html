Bootstrap Image Vertical Alignment with Text

📌 What is Vertical Alignment with Text?
    It means aligning image relative to text vertically:
    - Top aligned
    - Middle aligned
    - Bottom aligned
    - Perfect center (recommended)

📌 Important:
    Bootstrap uses FLEXBOX for proper vertical alignment.

1️⃣ Image Top Aligned with Text

📌 Concept
    align-items-start
    Image and text start from top

This paragraph starts from the top along with the image. This alignment is useful when you want image and text to begin from the same top position.

2️⃣ Image Vertically Center with Text (MOST USED)

📌 Concept
    d-flex → enable flexbox
    align-items-center → vertical center

This is the MOST COMMON real-world use case. The image stays perfectly centered relative to the text. Used in cards, list items, profile sections.

3️⃣ Image Bottom Aligned with Text

📌 Concept
    align-items-end
    Image and text align at bottom

Both image and text align at the bottom. This is rarely used but useful in some UI layouts.

4️⃣ Inline Image Vertical Alignment (OLD but useful)

📌 Concept
    vertical-align works with inline / inline-block elements
    Values: top | middle | bottom

This image is vertically aligned in the middle of the text line. Useful for icons or small images inside paragraphs.

5️⃣ Image Vertical Center using Bootstrap Grid

📌 Concept
    row + align-items-center
    Best for responsive layouts

This is the BEST method for responsive layouts. Image and text stay vertically aligned across screen sizes. Highly recommended for production projects.
