Bootstrap Brake Point Example 

Use Bootstrap's responsive grid system to create a layout that adjusts based on the screen size. The example below demonstrates how to use different column classes for various breakpoints.
         1. xxl : Extra Extra Large devices (≥1400px)    : 1400px and up (Large Desktop)
         2. xl  : Extra Large devices (≥1200px)          : 1200px to 1399px (Large Desktop)
         3. lg  : Large devices (≥992px)                 : 992px to 1199px (Desktop)
         4. md  : Medium devices (≥768px)                : 768px to 991px (Small Laptop/Tablet)
         5. sm  : Small devices (≥576px)                 : 576px to 767px (Tablet)
         6. xs  : Extra small devices (<576px)           : 0px to 575px (Mobile)

Example : Text Alignment

1. Desktop : Center

2. Tabet : Left

3. Mobile : Right

lg -> xl -> xxl : lg support to lg+ layout automatically

Means : lg design support to lg+ layout automatically

sm -> md : md support to sm to md layout automatically ( not support to lg+ & xs )

Means : sm design support to sm+ layout automatically ( not support to lg+ & xs )

xs : xs support to XS layout automatically ( not support to sm+ & lg+ )

Means : xs design support to xs layout automatically ( not support to sm+ & lg+ )


Responsive Text Alignment Example 

        Explanation :
        1. text-lg-center  : Center align for lg and larger devices (≥992px)
        2. text-sm-end     : Right align for sm and md devices (≥576px and <992px)
        3. text-start      : Left align for extra small devices (<576px)
