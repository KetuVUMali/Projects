Bootstrap justify-content – Deep Explanation

📌 What is justify-content?
    justify-content is a FLEXBOX property.
    It controls HORIZONTAL alignment of items inside a flex container.

📌 IMPORTANT RULES
    ✔ justify-content works ONLY with:
        d-flex or d-inline-flex
    ✔ It works on the MAIN AXIS
    ✔ Default flex direction is ROW (horizontal)

📌 Syntax
    justify-content-{value}

9️⃣ Common Mistakes

❌ Using justify-content without d-flex
❌ Expecting vertical alignment in flex-row
❌ Forgetting flex-direction

✔ Solution:
   Always check:
   - d-flex
   - flex-row / flex-column
