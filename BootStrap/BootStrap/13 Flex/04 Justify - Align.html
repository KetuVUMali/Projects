justify-content vs align-items (Deep Logic)

        RULE 1: d-flex is mandatory
        RULE 2: justify-content → MAIN AXIS
        RULE 3: align-items     → CROSS AXIS
        RULE 4: Axis depends on flex-direction
    
        flex-direction: row; (default behavior of d-flex class)
            
            Main Axis  → Horizontal (left ↔ right)   :  It controls HOW FLEX ITEMS MOVE ALONG THE MAIN AXIS
                All classes : horizontal movement
                            justify-content-start : align left
                            justify-content-center : align center
                            justify-content-end : align right
                            justify-content-between : distribute space between items
                            justify-content-around : distribute space around items
                            justify-content-evenly : distribute space evenly between items
            
            Cross Axis → Vertical   (top ↕ bottom)   :  It controls HOW FLEX ITEMS MOVE ALONG THE CROSS AXIS
                All classes : Vertical movement
                            align-items-start   : align top
                            align-items-center  : align center
                            align-items-end     : align bottom
                            align-items-baseline : align text baseline
                            align-items-stretch : stretch to fill container
