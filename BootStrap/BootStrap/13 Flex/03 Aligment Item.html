Bootstrap align-items (Super Clear Learning)

📌 What is align-items?
    align-items controls VERTICAL alignment of FLEX items
    inside a FLEX container.

📌 IMPORTANT RULES (MUST READ)
    ✔ d-flex is REQUIRED
    ✔ align-items works on CROSS AXIS
    ✔ Default flex-direction = row
    ✔ Container MUST have height

STEP 1️⃣ : Without align-items (Default Behavior)

📌 Default align-items value = stretch
    Items stretch to full container height
