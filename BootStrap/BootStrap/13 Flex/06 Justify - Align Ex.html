Bootstrap Flex: Real-World Examples

📌 CONCEPT RECAP:
- justify-content → controls MAIN AXIS alignment
- align-items     → controls CROSS AXIS alignment
- d-flex is mandatory
- Axis depends on flex-direction

Example 1: Header Navbar Layout (justify-content-between + align-items-center)

📌 Real-World: Website Header
- Logo on left

- All items vertically centered
- Responsive-friendly

📌 KEY TAKEAWAYS:

📌 Always remember:
- justify-content → main axis
- align-items     → cross axis
- flex-direction changes axis
