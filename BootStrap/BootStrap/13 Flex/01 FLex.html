Bootstrap FLEX – Complete Guide

📌 What is Flex?
Flex (Flexible Box) is used to:
• Align items easily
• Control direction (row / column)
• Control spacing
• Make responsive layouts

📌 Bootstrap Flex = CSS Flexbox with ready classes

1️⃣ d-flex (Enable Flexbox)

        📌 d-flex
        • Converts element into flex container
        • All children become flex items
