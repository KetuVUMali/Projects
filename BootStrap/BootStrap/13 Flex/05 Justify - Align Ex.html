Bootstrap Flex: Real-World Combined Examples

📌 FLEX RULES
- d-flex is mandatory
- justify-content → main axis (horizontal by default)
- align-items     → cross axis (vertical by default)
- flex-direction can change axis (row / column)

Example 1: Centered Profile Section

Scenario:
- Profile image + text (vertical center)
- Horizontal center in page

- justify-content-center
- align-items-center

✅ Key Learnings:
1. Main axis → justify-content
2. Cross axis → align-items
3. flex-direction changes axes
4. Combine both for layout control
5. Works in almost all layouts: navbars, profile cards, footer, etc.
