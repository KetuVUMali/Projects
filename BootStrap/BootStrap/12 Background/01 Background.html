Bootstrap Background – Complete Guide

📌 Background utilities are used to:
   - Add colors
   - Add gradients
   - Control opacity
   - Improve UI look easily

1️⃣ Background Color Classes

📌 Syntax:
   bg-color-name

📌 Used for:
   Cards, sections, alerts, headers
bg-primary
bg-secondary
bg-success
bg-danger
bg-warning
bg-info
bg-light
bg-dark

2️⃣ Background + Text Color Combination

📌 Important:
   Always adjust text color for readability
bg-warning + text-dark
bg-dark + text-white
bg-primary + text-white

3️⃣ Background Gradient

📌 Class:
   bg-gradient

📌 Used for:
   Modern UI look
bg-primary + bg-gradient
bg-success + bg-gradient
