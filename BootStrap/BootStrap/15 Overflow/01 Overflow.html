Bootstrap Overflow – VISUAL Understanding

Each box has SAME height. Content is MORE than height. See how overflow behaves.

overflow-visible (default)
Container

This text goes outside the box. You can see content overflowing. Nothing is hidden.

overflow-hidden
Container

This text is CUT. Extra content is hidden. You cannot scroll.

overflow-auto
Container

This text creates scroll only when needed. Scrollbar appears automatically.

overflow-scroll
Container

Scrollbar is ALWAYS visible even if content fits.
