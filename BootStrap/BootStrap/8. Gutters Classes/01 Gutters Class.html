Gutter = Space between columns.
            Bootstrap gives special classes to increase or decrease the space (padding) between columns inside a row.
        
 Gap : The .g-* classes are used to set the gap (gutter) between rows and columns.
                    The gap is the space between the columns and rows in a grid layout.

            Example :
            1. g-0  : No gutter (0px)
            2. g-1  : Small gutter (4px) (0.25 rem)
            3. g-2  : Medium gutter (8px) (0.5 rem)
            4. g-3  : Large gutter (16px) (1 rem)
            5. g-4  : Extra large gutter (24px) (1.5 rem)
            6. g-5  : Extra extra large gutter (32px) (2 rem)
            7. gx-* → Horizontal space (left–right) between columns
            8. gy-* → Vertical space (top–bottom) between rows

            gx-1 : Set horizontal gutter (left-right) to 4px (0.25 rem)
            gx-2 : Set horizontal gutter (left-right) to 8px (0.5 rem)
            gx-3 : Set horizontal gutter (left-right) to 16px (1 rem)
            gx-4 : Set horizontal gutter (left-right) to 24px (1.5 rem)
            gx-5 : Set horizontal gutter (left-right) to 32px (2 rem)

            gy-1 : Set vertical gutter (top-bottom) to 4px (0.25 rem)
            gy-2 : Set vertical gutter (top-bottom) to 8px (0.5 rem)
            gy-3 : Set vertical gutter (top-bottom) to 16px (1 rem)
            gy-4 : Set vertical gutter (top-bottom) to 24px (1.5 rem)
            gy-5 : Set vertical gutter (top-bottom) to 32px (2 rem)
            


            gx : Manage Padding Left-Right ( Horizontal ) between columns
            gy : Manage Margin Top-Bottom ( Vertical ) between rows

        
            Note :
                    Padding (p-*) and margin (m-*) are spacing utility classes Not Gutter Classes.
                    Gutter (g-*, gx-*, gy-*) is a separate feature used only inside the Bootstrap grid system.
                    ❌ Padding classes are NOT gutter classes
                    ❌ Margin classes are NOT gutter classes
                    ✔ Gutter classes are only: g-*, gx-*, gy-*
