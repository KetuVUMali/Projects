Bootstrap Position – Complete Concepts

📌 Position controls WHERE an element stays on the page.
📌 Bootstrap provides utility classes instead of custom CSS.
📌 Common real-world uses:
✔ Navbar fixed
✔ Badges on cards
✔ Floating buttons
✔ Overlays
✔ Sticky headers

1️⃣ Position Types

Classes:
.position-static     → Default (normal flow)
.position-relative   → Relative to itself
.position-absolute   → Relative to nearest positioned parent
.position-fixed      → Fixed to viewport
.position-sticky     → Sticks on scroll
position-static (default behavior)
position-relative (used as parent)

2️⃣ Absolute Position (with Relative Parent)

RULE:
✔ Absolute element looks for nearest relative parent
✔ Very common for badges, icons, overlays
Parent (position-relative) Top Left Top Right Bottom Left Bottom Right

3️⃣ Fixed Position

.position-fixed
✔ Stays at same place on screen
✔ Used for chat buttons, back-to-top, fixed navbar

4️⃣ Sticky Position

.position-sticky
.top-0
✔ Behaves like relative until scroll
✔ Then sticks to given position
✔ Used for headers, filters

🔟 Common Rules & Mistakes

❌ Absolute without relative parent → breaks layout
❌ Overusing fixed → bad UX
✔ Always use translate-middle for perfect center
✔ Sticky needs top-* to work
✔ Z-index works only on positioned elements
