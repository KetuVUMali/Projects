JavaScript Class

Class Definition

In JavaScript, a class is a blueprint for creating objects with predefined properties and methods. It provides a way to create and organize code in an object-oriented manner.

Usage of Class

Let's create a simple class named Person with properties name and age. We'll also add a method to display information about the person. 

        
        class Person 	 							// Class declaration
        {
			var name = "Initial Name";				 // var assigen
            constructor(name, age, abc) 			// constructo
	            {
	                this.name = name;
	                this.age = age;
	            }
            displayInfo()  					   		 // method
	            {
	                console.log(`Name: ${this.name}, Age: ${this.age}`);
	            }
        }

        let person1 = new Person("John Doe", 25, "ABC Value");			// person1 obj is create & pass para
		let person2 = new Person("Vilas", 50, "ABC Value");				// person2 obj is create & pass para
        person1.displayInfo();  										//obj through method call
		person2.displayInfo(); 											//obj through method call

Class Features

Let's explore various features of classes including getters, setters, constructors, inheritance (extends), and other class-related keywords. 

        // Base class
        class Animal 
        	{
        		// costructor
	            constructor(name) 
		            {
		                this.name = name;
		            }

	            // Getter (to get value)
	            get getName() 
		            {
		                return this.name;
		            }

	            // Setter (to set value) 
	            set setName(newName) 
		            {
		                this.name = newName;
		            }

	            // Method
	            makeSound() 
		            {
		                console.log("Generic animal sound");
		            }
	        }

        // Inherited class (extends)
        class Dog extends Animal {
            constructor(name, breed) // Constructor
	            {
	                // Super keyword is used to call the constructor of the parent class
	                super(name);
	                this.breed = breed;
	            }

            // Overriding method
            makeSound() 
	            {
	                console.log("Woof! Woof!");
	            }

            // New method specific to Dog class
            displayBreed() 
	            {
	                console.log(`Breed: ${this.breed}`);
	            }
        }

        // create class object
        let genericAnimal = new Animal("Generic Animal");
        console.log("Getter - Name:", genericAnimal.getName);

        genericAnimal.setName = "Updated Animal";
        console.log("Setter - Updated Name:", genericAnimal.getName);

        genericAnimal.makeSound();

        let dog1 = new Dog("Buddy", "Golden Retriever");
        dog1.makeSound();
        dog1.displayBreed();
