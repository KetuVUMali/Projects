Array 

		-Array is group of continuous memory locations.(memory address)
		-It is used to store multiple values into Single variable.
		-Index start from Zero
		- Array in JS are store multi data Types 
		- data store in array are key : value pair (key : index no, & value = ele in array)
		- Array size run time set
