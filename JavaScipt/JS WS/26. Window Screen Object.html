Window Screen Object 

		
		- The Screen Object Contains Information About THe Visitors Screen.

		- show the user screen information or size.
	
		Property :

			1. availHeight    : Return Height of Screen 
			  Syn : screen.availHeight

			2. availwidth     : Return width of screen.
			  Syn : screen.availwidth

			3. colorDepth      : Return the bit depth of the color palette for displaying images.
			  Syn : screen.colorDepth

			4. height         : return the total height ofscreen.
			  Syn : screen.height

			5. pixelDepth     : Return the color resolution (in bit per pixel) of the screen.
			  Syn : screen.pixelDepth

			6. width          : Return thr total width of screen.
			  Syn : screen.width
