Exception Handling 

		 Exception :
		 			1. Exception is abnormal Condition.

		 			2. Exception means handle error

		 			3. Exception handling is a process or method used for handling the abnormal ststements in the code and executing them.

		 			4. Try, Catch, throw, finally are used.



		
				try{ } : 
						 -The code which needs possible error testing is kept within the try block.

						 - In case any error occure in this try block then it pass catch{} block for handle error.


				catch{ } : 
						  - This block handle the error.


				syntax : 
								try
									{
										//
									}
								catch(error)
									{
										//
									}
