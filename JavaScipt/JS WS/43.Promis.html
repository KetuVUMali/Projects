JavaScript Promise (Easy Explanation)
        - A Promise is a special object in JavaScript.
        - It helps you handle things that take time, like loading data.
        - A Promise can be in one of three states:
            1. pending: still waiting for the result.
            2. fulfilled: finished successfully.
            3. rejected: something went wrong.
        - You use .then() to do something when it finishes successfully.
        - You use .catch() to handle errors if something goes wrong.

        Example
                let p = new Promise(function(resolve, reject) {
                // doing something that takes time, like loading a file
                let fileLoaded = true; // this is just an example
                if (fileLoaded) {
                    resolve("File loaded successfully!"); // promise fulfilled
                } else {
                    reject("Error loading file."); // promise rejected
                }
                });

                p.then(function(message) {
                console.log(message); // "File loaded successfully!"
                }).catch(function(error) {
                console.log(error); // "Error loading file."
                });
