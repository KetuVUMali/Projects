setInterval() 

        - Perform perticular task repeatedly some time
        - The setInterval() method repeats a given function at every time given time interval.
         (interval : repeatation)

        - Repeat perticular task or operation.
        
        - Syntax : window.setInterval(function,milli seconds);

        - function : first parameter is the function to be executed.

        - milliseconds : indicates the length of the time - interval between each execution.

        - continuously repeat task , not stop. (Continous perfom task)
