Attribute 

		- Attribute means we pass any type of class for any tag.

		- Ex. id="ketu" , class="ketu", title, name, alt, etc.

		- In this example id & class is attribute and ketu is attribute value.

		-  id="ketu"; (attribute="value";)

		-  onclick="testing()"  (event_name="function_name_()")

getAttribute()

 
				- to get attribute value.
				- to used get/collect class(attribute) value.

				- syn:
				      .getattribute("class/id_name");

setAttribute()

			- to set attribute value.
			- to used set/change class(attribute) value.

			- syn: (2 value pass. 1 is which class to set value & 2 is set value)
				      .setattribute("class/id_name","value");
