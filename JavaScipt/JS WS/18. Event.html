Event 

		- In programming, event is an occurrence of any action.

		- When we use JavaScript in HTML pages,JS can react with events.

		- Ex. When use click the Mouse button, or press any key on the keyboard.
	
		
 Mouse Event


		- onclick : The Event occurs when the user click on an element.

		- oncontexmenu : User right-clicks on an element to open a context menu.

		- ondblclick - The user double (dbl)click on an elemrent

		- onmousedown : user presses a mouse button over an element.

		- onmouseup : the user releases a mouse button while over an element. 

		- onmouseenter : The pointers moves out of element.

		- onmouseleave : Pointer move out of an element.

		- onmousemove : THe pointer is moving while it is over an element.

		- onmouseover : when the pointer is moved onto an element or one of its children.

		- onmouseout : user moves the mouse pointer out of an element or one of its children.
	
		
 Keyboard Event


		- onkeydown : When the user is pressing a key down.(press Key)

		- onkeypress : The movement the user starts pressing a key.(release pree key)

		- onkeyup - The user releases a key. (Combine onkeydown & onkeypress)
