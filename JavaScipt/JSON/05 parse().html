A common use of JSON is to exchange data to/from a web server.

        When receiving data from a web server, the data is always a string.

        Parse the data with JSON.parse(), and the data becomes a JavaScript object.



        JSON.parse() is a built-in JavaScript function that converts a JSON string into a JavaScript object.

        JSON.parse() to convert text into a JavaScript object





        JSON.parse(): Converts a JSON string into a JavaScript object or array.
        Example: JSON.parse('{"name": "Ketan"}') → {name: "Ketan"}

        parseInt(): Converts a string to an integer.
        Example: parseInt("123") → 123

        parseFloat(): Converts a string to a floating-point number.
        Example: parseFloat("123.45") → 123.45

        Number(): Converts a value to a number (more strict than parseInt()).
        Example: Number("123.45") → 123.45

        Date.parse(): Converts a date string into a timestamp (milliseconds since 1970).
        Example: Date.parse("2024-10-17T12:00:00Z") → 1729051200000

        split(): Breaks a string into an array based on a delimiter (useful for parsing strings).
        Example: "apple,banana,cherry".split(",") → ["apple", "banana", "cherry"]
