String : String = text / characters. A string is a sequence of characters enclosed in single quotes (' '), double quotes (" "), or backticks (` `). Strings are used to represent text and can contain letters, numbers, symbols, and whitespace. Syntax: let variableName = "value"; Example: let name = "John"; // double quotes let message = 'Hello, World!'; // single quotes let greeting = `Welcome to JavaScript!`; // backticks (template literals) Template Literals : A way to embedded expression in string. Template literals are a way to create strings that can contain embedded expressions. They are enclosed in backticks (` `) and allow for multi-line strings and string interpolation. means we can include variables and expressions inside a string without needing to concatenate them. expressions : A combination of variables, operators, and values that can be evaluated to produce a result. example : 2 + 3, name.toUpperCase(), etc. Syntax: let variableName = `string with ${expression}`; Example: 1. String Interpolation:s let name = "Alice"; let age = 30; let message = `My name is ${name} and I am ${age} years old.`; console.log(message); // Output: My name is Alice and I am 30 years old. 2. Numeric Expressions in Template Literals: let a = 5; let b = 10; let result = `The sum of ${a} and ${b} is ${a + b}.`; console.log(result); // Output: The sum of 5 and 10 is 15. console.log("The value of 2 + 3 is", 2 + 3); // Output: The value of 2 + 3 is 5. console.log(`The value of 2 + 3 is ${2 + 3}.`); // Output: The value of 2 + 3 is 5. 3. String Length in Template Literals: let str = "Hello"; let message = `The length of the string "${str}" is ${str.length}.`; console.log(message); // Output: The length of the string "Hello" is 5. 4. Using Methods in Template Literals: let name = "Bob"; let greeting = `Hello, ${name.toUpperCase()}!`; console.log(greeting); // Output: Hello, BOB! 5. Objects in Template Literals: let object = { name: "Charlie", age: 25 }; let message = `The person's name is ${object.name} and they are ${object.age} years old.`; console.log(message); // Output: The person's name is Charlie and they are 25 years old.
