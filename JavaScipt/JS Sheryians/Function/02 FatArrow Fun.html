Why Use Arrow Functions?
                ✅ Shorter and cleaner code
                ✅ No need for the function keyword
                ✅ Automatically binds 'this' (important for objects & event handlers)


            When to Use Arrow Functions?
                ✔ When using short and simple functions
                ✔ When you don't need this binding
                ✔ When working with array methods like map(), filter(), reduce()

            When NOT to Use Arrow Functions?
                ❌ When you need this to refer to a specific object (e.g., in class methods)
                ❌ When defining a method inside an object (this won't work properly)
                ❌ When using as a constructor function (new won’t work)


            🚀 Arrow functions make JavaScript functions shorter and cleaner. 
                They are useful for callbacks, array methods, and maintaining this scope in objects. 
                However, they are not suitable for all cases, especially when you need this to refer to a specific object.
